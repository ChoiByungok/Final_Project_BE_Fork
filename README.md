# 내가 맡은 기능
## 상품도메인
 queryDSL을 사용하여 동적쿼리를 생성해내고 상품 필터링을 담당함
 또한 상품 상세정보 페이지에 진입하기 위해 3개의 테이블을 조인해야 하는 상황에서 N+1문제가 안터지도록 로직을 짬
## 예약도메인
 하나의 예약에서 여러개의 상품을 받을 수 있도록 중간에 매핑테이블을 두어서 다대다 관계를 풀어냄
 다양한 예외처리를 구현하여 최대한 보안에 신경을씀
